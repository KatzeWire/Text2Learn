Next Steps

Expand On What We Have

We are always looking to add to the features and services provided in Text2Learn. This would include adding a Learn Definition feature where a student will be able to text us a word and we would respond with the definition and parts of speech for the word. Something else we are looking to include in our app is the ability to learn other common languages such as Learn Italian, Learn French, etc. With these other languages available to students, we would also provide the Learn Definition feature.

Add More Subjects

Something we are hoping to add to Text2Learn is the ability to learn the basics Computer Science. Some of the courses under Computer Science that we would offer would be different languages (Learn Java, Learn Python, Learn C, etc.). However, due to our limitation of being a solely SMS-based product, we would not be able to have students complete major programming projects, but instead would have our students do simple tasks. These would include everything ranging from how to set variables to performing basic math operations to correcting syntax for every language.
